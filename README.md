# LSTM-and-Georgia-Crop-Forecasting

A LSTM deep learning model that can imitate crop yield and irrigation demand modeling/prediction of DSSAT based on bi-weekly climate data of TMAX, TMIN, SRAD, RAIN during the growing season (16 two-week periods).
